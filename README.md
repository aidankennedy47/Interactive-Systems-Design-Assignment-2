# Leaflog-Dashboard instructions
### To run the webpage
1. Ensure you have NodeJS installed by typing "node -v" and "npm -v" into your terminal. (should pop up with v24.x.x and 11.x.x as of June 2026)
1a. If you do not have NodeJS installed, install the LTS package from https://nodejs.org/.
2. Install the files from this repo through this release https://github.com/aidankennedy47/Interactive-Systems-Design-Assignment-2/releases/tag/Prototype
3. Unzip the files and open the leaflog-dashboard folder through a terminal program (VSC terminal, terminal or powershell)
4. Run "npm install" to install necessary files and then run "npm start" to start the website.
